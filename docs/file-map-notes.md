## 🚀 快速启动指南

### 1. **激活ROS2环境**
```bash
# SSH中执行
source /opt/ros/humble/setup.bash
cd /home/racecar
source install/setup.bash
```

### 2. **启动小车基础功能**
```bash
# 启动传感器
ros2 launch racecar sensor.launch.py

# 启动导航系统
ros2 launch racecar nav.launch.py

# 启动巡线功能
python3 src/racecar/scripts/detct.py
```

### 3. **启动AI追踪功能**
